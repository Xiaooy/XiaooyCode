#### 一、删除node_modules文件夹
1. 方法一：
```
$ npm install rimraf -g
$ rimraf node_modules
```
1. 方法二：
 ```
$ rmdir /s/q your_app_dir
```
1. 方法三：
 ```
$ rm -f /node_modules
```
#### 二、添加node_modules文件夹
```
$ npm install
```
